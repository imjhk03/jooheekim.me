# 언어 전환 테스트

이것은 언어 전환 기능이 제대로 작동하는지 확인하기 위해 **한국어**로 작성된 테스트 포스트입니다.

## 기능

- 이중 언어 콘텐츠 지원
- 쉬운 언어 전환
- 깔끔한 사용자 인터페이스

위에 English와 한국어 옵션이 표시된 언어 전환기가 보일 것입니다.
