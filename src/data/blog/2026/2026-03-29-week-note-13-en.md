Mar 23-29, 2026

I safely shipped my personal app. Since it had been a while since the last update, I moved a little too quickly, and that left me with some lingering regret. After I started my contract project, work on the personal app kept stopping and starting again, so it often felt like I had lost the flow. Because of that, it took longer than I expected to review everything and fix bugs. I also had to test with real data, which made the work more involved than I had planned. Still, I feel like I'm gradually finding an AI programming workflow that fits me, so maybe the next update will go a little more smoothly.

Most of the major features for the contract project are done now, so this week I focused on smaller supporting tasks. That mostly meant cleaning up unused code and improving things that tend to get missed during development. In the process, I'm getting more familiar with the codebase too. Personally, I don't think I would have embraced AI programming this actively if I hadn't taken on this contract project. It gave me enough flexibility to experiment with different ways of working with AI while building the project. In the end, it was work I had to do for income, but I still feel like it was the right decision.

I finally updated my M1 Pro MacBook Pro to macOS Tahoe. It took a while because I needed to get my personal project into a safe stopping point first, but I finally did it. The design will take some getting used to, but I like that trying the new features already makes me want to build new things. I already have a few new app ideas in mind, and I plan to develop them after refining them a little more. I hope I can keep turning the things I've been thinking about into something real, little by little.

The weather has warmed up a lot. More flowers are blooming, and fresh shoots are coming up from the ground. Since the temperature already feels similar to April last year, part of me thinks it will be a problem if it gets this warm this early, but part of me also feels like this is just something I have to endure. March is coming to an end. The first quarter of 2026 is already passing by. I want to do well again in the next quarter.
